# 🎻 Orquestra Jovem Cidadão - Landing Page

Esta é a landing page oficial do projeto social **Orquestra Jovem Cidadão**. O objetivo principal da página é dar visibilidade à orquestra, divulgar as ações musicais e comunitárias, e conectar o projeto com apoiadores e futuros alunos.

---

## 🚀 Tecnologias Utilizadas

O projeto foi construído utilizando tecnologias web fundamentais, focando em um design responsivo, limpo e acessível:

- **HTML5:** Estruturação semântica do conteúdo.
- **CSS3:** Estilização moderna, utilizando Flexbox para a organização do layout e responsividade.
- **JavaScript (Vanilla):** Implementação de interações dinâmicas e comportamento da interface.

---

## 💻 Funcionalidades da Página

- **Design Responsivo:** Adaptado perfeitamente para visualização em smartphones, tablets e computadores.
- **Seção Sobre o Projeto:** Detalhes sobre a história, missão e o impacto social da orquestra através da educação musical.
- **Galeria / Eventos:** Espaço dedicado para exibir as apresentações e a evolução dos alunos.
- **Formulário de Contato / Inscrição:** Canal direto para quem deseja apoiar, fazer parcerias ou se inscrever nas aulas.
